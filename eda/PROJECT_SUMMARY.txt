📞 SUPPORT & TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────

ISSUE: "ModuleNotFoundError"
  → Run: pip install -r requirements_eda.txt

ISSUE: "File not found" for rasters
  → Verify paths in Section 1 of notebook
  → Check files exist at d:\sindupalchok_landslide\data\processed\

ISSUE: "Memory error"
  → Close other applications
  → Ensure 4+ GB free RAM
  → Try on different computer

ISSUE: "GDAL error"
  → Install: pip install --upgrade gdal
  → Or: conda install gdal

ISSUE: Landslide points outside boundary
  → Check CRS alignment
  → Verify coordinate format
  → Check for coordinate sign errors

ISSUE: Notebook runs slow
  → Normal! Value extraction takes time
  → Expected: 5-8 minutes total
  → Be patient 😊
